# 🎮 Pong Game

一个经典的 Pong 游戏实现，使用纯 HTML5、CSS3 和 JavaScript 开发。在这个游戏中，玩家需要与计算机 AI 对战，通过控制球拍来击球得分。

## ✨ 游戏特性

### 核心功能
- 🎯 **对战模式**：与智能 AI 计算机对手竞技
- 🎨 **现代美观的界面**：渐变背景与流畅的动画效果
- 📊 **实时计分**：清晰显示玩家和计算机的得分
- ⚙️ **AI 智能算法**：计算机对手会追踪球的位置自动移动
- 🎚️ **4 级难度选择**：简单、中等、困难、极难

### 游戏机制
- 🎪 **球的物理效应**：
  - 碰撞墙壁时反弹
  - 击中球拍时会产生"旋转"效果
  - 球速根据难度动态调整
  
- 🕹️ **灵活的控制方式**：
  - **鼠标控制**：在游戏区域移动鼠标，球拍会跟随移动（支持上下左右）
  - **键盘控制**：
    - `↑` / `↓` 箭头键或 `W` / `S` 上下移动球拍
    - `←` / `→` 箭头键或 `A` / `D` 左右移动球拍

- 🎮 **游戏控制**：
  - **Start/Pause** 按钮：开始游戏或暂停游戏
  - **Reset Score** 按钮：重置计分并重新开始
  - **更换难度** 按钮：随时切换 AI 难度等级

### 难度等级 🎯
游戏提供 4 个难度等级，可在游戏开始或中途随时更换：

| 难度 | AI 速度 | 球速 | 特点 |
|------|--------|------|------|
| 🟢 简单 | 3 | 0.8x | 反应慢，适合新手练习 |
| 🔵 中等 | 5 | 1.0x | 平衡难度，默认选项 |
| 🟠 困难 | 7 | 1.2x | 反应快，有挑战性 |
| 🔴 极难 | 9 | 1.5x | 反应极快，几乎无敌 |

## 📋 游戏规则

1. 选择喜欢的难度等级后开始游戏
2. 玩家控制**右侧球拍**，AI 控制**左侧球拍**
3. 当球穿过对手时，对方得 1 分
4. 游戏可以无限进行，尽量积累更高的分数
5. 点击"Reset Score"重置游戏，"更换难度"可以切换难度等级

## 💻 技术栈

- **HTML5**：游戏结构和画布渲染
- **CSS3**：现代化界面设计和动画效果
- **JavaScript (Vanilla)**：游戏逻辑、物理引擎和难度系统
- **Canvas API**：图形绘制和动画

## 🎮 快速开始

### 方式 1：在线游玩（推荐）
直接访问：https://7phw.github.io/pong-game/

### 方式 2：本地运行

1. **克隆仓库**
```bash
git clone https://github.com/7PHW/pong-game.git
cd pong-game
```

2. **打开游戏**
   - 方式 A：直接在浏览器中打开 `index.html` 文件
   - 方式 B：使用本地服务器（推荐）
   ```bash
   # 如果安装了 Python 3
   python -m http.server 8000
   
   # 如果安装了 Node.js
   npx http-server
   ```
   然后在浏览器中访问 `http://localhost:8000`

3. **开始游戏**
   - 选择喜欢的难度等级
   - 点击"Start Game"按钮开始游戏
   - 使用鼠标或键盘控制球拍
   - 尽力击败 AI 对手！

## 🎯 游戏攻略

### 基础技巧
- 🎪 **反弹技巧**：球击中球拍的不同位置会产生不同的旋转效果
- ⚡ **位置预判**：提前移动球拍到球将要到达的位置
- 🛡️ **防守策略**：保持球拍在球的正前方可以提高防守效率

### 难度提升建议
- 👶 **新手**：从"简单"难度开始适应游戏节奏
- 🎯 **进阶**：升级到"中等"难度提升技术
- 🔥 **高手**：挑战"困难"和"极难"难度
- 💪 **专家**：尝试在"极难"难度下保持连胜

## 🔧 自定义设置

### 修改游戏参数
你可以在 `script.js` 文件中修改以下参数来自定义游戏：

```javascript
// 基础参数
const paddleWidth = 10;           // 球拍宽度
const paddleHeight = 80;          // 球拍高度
const ballSize = 8;               // 球的大小
const basePlayerSpeed = 6;        // 玩家球拍基础速度
```

### 添加新难度等级
编辑 `difficultyLevels` 对象即可添加自定义难度：

```javascript
const difficultyLevels = {
    easy: {
        aiSpeed: 3,
        ballSpeedMultiplier: 0.8,
        label: '简单'
    },
    // ... 其他难度
    custom: {
        aiSpeed: 6,
        ballSpeedMultiplier: 1.1,
        label: '自定义'
    }
};
```
